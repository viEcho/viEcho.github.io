#### 安装java
查看yum库中的Java安装包：
`yum -y list java*`
将java-1.8.0的所有相关Java程序都安装上：
`yum -y install java-1.8.0-openjdk*`

查看刚安装的Java版本信息。
输入：java -version 可查看Java版本
![](https://s2.ax1x.com/2020/01/17/1SYzVO.png)

输入：javac 可查看Java的编译器命令用法
![](https://s2.ax1x.com/2020/01/17/1StSaD.png)
