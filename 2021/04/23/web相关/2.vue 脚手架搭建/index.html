1пјҡе®үиЈ…node

з«ҜејҖеҸ‘жЎҶжһ¶е’ҢзҺҜеўғйғҪжҳҜйңҖиҰҒ Node.js пјҢе…Ҳе®үиЈ…node.jsејҖеҸ‘зҺҜеўғпјҢvueзҡ„иҝҗиЎҢжҳҜиҰҒдҫқиө–дәҺnodeзҡ„npmзҡ„з®ЎзҗҶе·Ҙе…·жқҘе®һзҺ°пјҢдёӢиҪҪhttps://nodejs.org/en/пјҢе®үиЈ…е®ҢжҲҗд№ӢеҗҺпјҢжү“ејҖcmdејҖе§Ӣиҫ“е…Ҙе‘Ҫд»ӨгҖӮпјҲжҲ‘з”Ёзҡ„жҳҜwin10зі»з»ҹпјҢжүҖд»ҘйңҖиҰҒз®ЎзҗҶе‘ҳжқғйҷҗпјҢеҸій”®зӮ№еҮ»д»Ҙз®ЎзҗҶе‘ҳиә«д»ҪиҝҗиЎҢcmdпјүпјҢдёҚз„¶дјҡеҮәзҺ°еҫҲеӨҡжҠҘй”ҷгҖӮ

2пјҡжҹҘзңӢnodeзҡ„зүҲжң¬еҸ·

дёӢиҪҪеҘҪnodeд№ӢеҗҺпјҢд»Ҙз®ЎзҗҶе‘ҳиә«д»Ҫжү“ејҖcmdз®ЎзҗҶе·Ҙе…·пјҢпјҢиҫ“е…Ҙ node -v пјҢеӣһиҪҰпјҢжҹҘзңӢnodeзүҲжң¬еҸ·пјҢеҮәзҺ°зүҲжң¬еҸ·еҲҷиҜҙжҳҺе®үиЈ…жҲҗеҠҹгҖӮ

иҫ“е…Ҙе‘Ҫд»Өпјҡ node -v

3пјҡе®үиЈ…ж·ҳе®қnpmй•ңеғҸ

з”ұдәҺnpmжҳҜеӣҪеӨ–зҡ„пјҢдҪҝз”Ёиө·жқҘжҜ”иҫғж…ўпјҢжҲ‘д»¬иҝҷйҮҢдҪҝз”Ёж·ҳе®қзҡ„cnpmй•ңеғҸжқҘе®үиЈ…vue.
ж·ҳе®қзҡ„cnpmе‘Ҫд»Өз®ЎзҗҶе·Ҙе…·еҸҜд»Ҙд»Јжӣҝй»ҳи®Өзҡ„npmз®ЎзҗҶе·Ҙе…·гҖӮ
иҫ“е…Ҙе‘Ҫд»Өпјҡnpm install -g cnpm --registry=https://registry.npm.taobao.org

4пјҡе®үиЈ…е…ЁеұҖvue-cliи„ҡжүӢжһ¶

ж·ҳе®қй•ңеғҸе®үиЈ…жҲҗеҠҹд№ӢеҗҺпјҢжҲ‘д»¬е°ұеҸҜд»Ҙе…ЁеұҖvue-cliи„ҡжүӢжһ¶пјҢиҫ“е…Ҙе‘Ҫд»Өпјҡcnpm install вҖ“global vue-cli еӣһиҪҰпјӣйӘҢиҜҒжҳҜеҗҰе®үиЈ…жҲҗеҠҹпјҢеңЁе‘Ҫд»Өиҫ“е…ҘvueпјҢеҮәжқҘvueзҡ„дҝЎжҒҜпјҢеҸҠиҜҙжҳҺе®үиЈ…жҲҗеҠҹпјӣ
иҫ“е…Ҙе‘Ҫд»Өпјҡcnpm install --global vue-cli

5пјҡе»әдёҖдёӘж–°йЎ№зӣ®

жҗӯе»әе®ҢжүӢи„ҡжһ¶д№ӢеҗҺпјҢжҲ‘д»¬иҰҒејҖе§Ӣе»әдёҖдёӘж–°йЎ№зӣ®пјҢиҝҷдёӘж—¶еҖҷжҲ‘е»әи®®пјҢе°ҪйҮҸдёҚиҰҒиЈ…еңЁCзӣҳпјҢеӣ дёәvueдёӢиҪҪдёӢжқҘзҡ„ж–Үд»¶жҜ”иҫғеӨ§пјҢеҰӮжһңиҰҒж”№зӣҳзҡ„иҜқпјҢзӣҙжҺҘиҫ“е…ҘDпјҡеӣһиҪҰе°ұеҸҜд»ҘзӣҙжҺҘж”№зӣҳ
иҫ“е…Ҙе‘Ҫд»Өпјҡvue init webpack my-project-first
еӣһиҪҰпјҢmy-project-firstжҳҜжҲ‘иҮӘе·ұзҡ„ж–Үд»¶еӨ№зҡ„еҗҚеӯ—пјҢжҳҜеҹәдәҺwebpackзҡ„йЎ№зӣ®пјҢиҫ“е…Ҙд№ӢеҗҺе°ұдёҖзӣҙеӣһиҪҰпјҢзӣҙеҲ°еҮәзҺ°жҳҜеҗҰиҰҒе®үиЈ…vue-routeпјҢ
иҝҷдёӘжҲ‘д»¬еңЁйЎ№зӣ®иҰҒз”ЁеҲ°пјҢжүҖд»Ҙе°ұиҫ“е…Ҙy еӣһиҪҰ

6пјҡжіЁж„Ҹ

дёӢйқўдјҡеҮәзҺ°жҳҜеҗҰйңҖиҰҒjsиҜӯжі•жЈҖжөӢпјҢиҝҷдёӘжҲ‘д»¬жҡӮж—¶з”ЁдёҚеҲ°пјҢе°ұеҸҜд»ҘзӣҙжҺҘиҫ“е…ҘnoпјҢеҗҺйқўзҡ„йғҪеҸҜд»ҘзӣҙжҺҘиҫ“е…ҘnoпјҢйғҪжҳҜжҲ‘д»¬жҡӮж—¶з”ЁдёҚеҲ°зҡ„
жү“ејҖDзӣҳжҹҘзңӢдёҖдёӢпјҢдјҡеҸ‘зҺ°еӨҡдәҶдёҖдёӘеҲҡеҲҡеҲӣе»әзҡ„ж–Үд»¶еӨ№

7пјҡиҝӣе…ҘйЎ№зӣ®ж–Үд»¶еӨ№

ж–Үд»¶еӨ№е·Із»ҸдёӢиҪҪеҘҪдәҶпјҢзҺ°еңЁе°ұеҸҜд»Ҙиҝӣе…Ҙж–Үд»¶еӨ№пјҢиҫ“е…Ҙпјҡ cd my-project-first еӣһиҪҰиҝӣе…Ҙж–°е»әзҡ„йЎ№зӣ®гҖӮ

8пјҡеңЁйЎ№зӣ®йҮҢе®үиЈ…дҫқиө–

еӣ дёәеҗ„дёӘжЁЎжқҝд№Ӣй—ҙйғҪжҳҜзӣёдә’дҫқиө–зҡ„пјҢжүҖд»ҘзҺ°еңЁжҲ‘д»¬иҰҒе®үиЈ…дҫқиө–пјҢеңЁйЎ№зӣ®йҮҢиҫ“е…Ҙд»ҘдёӢе‘Ҫд»ӨгҖӮ
иҫ“е…Ҙе‘Ҫд»Өпјҡ cnpm install

9пјҡиҝҗиЎҢ

дёҖеҲҮзҺҜеўғдҫқиө–е®үиЈ…еҮҶеӨҮе°ұз»ӘпјҢжҲ‘д»¬жқҘжөӢиҜ•дёҖдёӢиҮӘе·ұж–°е»әзҡ„vueйЎ№зӣ®зҡ„иҝҗиЎҢжғ…еҶөпјҢиҫ“е…Ҙе‘Ҫд»Өпјҡcnpm run devзӣҙжҺҘеӣһиҪҰгҖӮдјҡеј№еҮәдёҖдёӘжөҸи§ҲеҷЁи®ҝй—®ең°еқҖй»ҳи®Өдёәlocalhost:8080гҖӮпјҲжҲ‘е°Ҷең°еқҖдҝ®ж”№дёә8081иҝӣиЎҢи®ҝй—®пјү
иҫ“е…Ҙе‘Ҫд»Өпјҡcnpm run dev

10пјҡеңЁжөҸи§ҲеҷЁиҫ“е…Ҙlocalhost:8081

8080жҳҜй»ҳи®Өзҡ„з«ҜеҸЈпјҢиҰҒи®ҝй—®зҡ„иҜқпјҢзӣҙжҺҘеңЁжөҸи§ҲеҷЁиҫ“е…Ҙlocalhost:8080е°ұеҸҜд»Ҙжү“ејҖй»ҳи®Өзҡ„жЁЎжқҝдәҶпјӣпјҲжҲ‘зҡ„з”өи„‘дёҠ8080з«ҜеҸЈжңүйңҖиҰҒпјҢиў«еҸҰеӨ–дёҖдёӘйЎ№зӣ®еҚ з”ЁпјҢж•…иҖҢдҪҝз”Ё8081зҡ„з«ҜеҸЈпјү
еңЁжөҸи§ҲеҷЁиҫ“е…Ҙlocalhost:8081пјҢжҳҫзӨәеҰӮдёӢпјҢеҲ°жӯӨдёәжӯўпјҢvueејҖеҸ‘зҺҜеўғжҗӯе»әе®ҢжҜ•гҖӮ
дҪҶжҳҜеңЁжңҖеҗҺдёҖжӯҘзҡ„ж—¶еҖҷпјҢжңүдәӣдәәдјҡйҒҮеҲ°иҝҷж ·зҡ„жҠҘй”ҷпјҡ

дёҚиҰҒзқҖжҖҘпјҢжҲ‘д№ҹйҒҮеҲ°иҝҮиҝҷж ·зҡ„жғ…еҶөпјҢжҖ»з»“дәҶдёҖдёӘйқһеёёе®ҢзҫҺзҡ„и§ЈеҶіж–№жЎҲгҖӮ
и§ЈеҶіж–№жЎҲпјҡ

1пјҡеңЁйЎ№зӣ®йҮҢжү§иЎҢиҝҷж ·зҡ„е‘Ҫд»Ө

npm i prettier@~1.12.0

2пјҡйҮҚж–°ејҖеҗҜиҝҗиЎҢеҚіеҸҜжҲҗеҠҹпјҡ

npm run dev

е°Ҹз»“пјҡжңҖеҗҺеҰӮдҪ•дҝ®ж”№й»ҳи®Өзҡ„з«ҜеҸЈпјҹ
ж—ўз„¶жҸҗеҲ°иҮӘе·ұдҝ®ж”№дәҶй»ҳи®Өзҡ„з«ҜеҸЈпјҢйӮЈе°ұиҜҙжҳҺдёҖдёӢ:
ж №зӣ®еҪ•дёӢжңүдёҖдёӘconfigж–Үд»¶еӨ№пјҢзңӢеҗҚеӯ—е°ұзҹҘйҒ“дёҺй…ҚзҪ®жңүе…іпјҢжү“ејҖconfigзӣ®еҪ•дёӢзҡ„index.jsпјҢиҝҷйҮҢдёҚд»…еҸҜд»Ҙж”№еҸҳз«ҜеҸЈпјҢиҝҳеҸҜд»Ҙж №жҚ®йңҖиҰҒж”№еҸҳе…¶д»–й…ҚзҪ®дҝЎжҒҜгҖӮ
